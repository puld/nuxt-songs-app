# Система Агентов для LLM - Быстрый старт

## Быстрое начало

### Шаг 1: Планирование
Запустите агента планирования для обсуждения вашей задачи:
```
/sc:brainstorm
```

Агент планирования:
1. Обсудит с вами требования
2. Проанализирует проект
3. Создаст детальный план
4. Сохранит контекст в память

### Шаг 2: Исполнение
Запустите агента исполнения для выполнения задач:
```
/sc:implement
```

Агент исполнения:
1. Загрузит план из памяти
2. Выполнит следующую задачу
3. Сохранит отчет о выполнении
4. Передаст следующую задачу

### Шаг 3: Повторение
Повторяйте шаг 2 для всех задач в плане.

## Команды управления контекстом

```bash
# Просмотр текущего плана
/claude:plan

# Просмотр лога сессий
/claude:log

# Сохранение контекста
/claude:save

# Очистка контекста (после завершения)
/claude:cleanup
```

## Требования

- Все агенты работают только на русском языке
- Используют MCP Serena для анализа и редактирования кода
- Не используют стандартные утилиты (sed, awk и др.) для изменения файлов
