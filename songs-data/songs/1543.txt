#1543 Для Бога жить – святое приглашенье

1. Для Бога жить – святое приглашенье,
Для Бога жить – то долг души,
/Для Бога жить – то призыв ко спасенью:
Для Бога жить, для Бога жить. /2р.

2. Для Бога жить – Христова Кровь лилася,
Чтоб грешных с Богом примирить,
/Чтоб каждый грешник мог Христу отдаться:
Для Бога жить, для Бога жить. /2р.

3. Для Бога жить, хоть мир враждой встречает,
И льётся кровь, и смерть грозит –
/То жребий всех, кто пламенно желает
Для Бога жить, для Бога жить. /2р.

4. Для Бога жить, ведь Он не оставляет
Того, себя кто не щадит,
/Но твёрдо лишь одно в душе решает:
Для Бога жить, для Бога жить! /2р.

5. Для Бога жить – пускай волной последней
Пройдёт по миру этот зов.
/Для Бога жить, нельзя уж больше медлить:
Христос грядёт, Христос грядёт! /2р.
