# Исправление переполнения заголовков песен

## Анализ причин

### 1. Заголовок в навбаре (`layouts/default.vue`) — **ОСНОВНАЯ ПРОБЛЕМА**

- `.app-bar` имеет фиксированную `height: 56px` — когда заголовок переносится на 2+ строки, он выходит за пределы по вертикали. В сочетании с `align-items: center` вторая/третья строка вылезает вниз, а верх первой строки обрезается по верхней границе 56px контейнера.
- `.nav-title` имеет `font-size: 2rem` (~32px) — слишком крупно для мобильных. Типичный заголовок вроде «1254. Хотела б я петь о любви Христа» на ~200px свободного места между двумя кнопками навбара переносится на 3 строки.
- `#navbar-center` **вообще без стилей** — нет `max-width`, нет `overflow: hidden`, нет обрезания текста.
- `.page-content` имеет конфликт отступов: сначала `padding-top: 56px`, затем перезаписывается `padding: 1rem` — верхний отступ теряется (мелкий баг).

### 2. Переполнение заголовков в поисковой выдаче (`pages/index.vue`)

- `.song-title` имеет `flex-grow: 1`, но без `overflow` и `text-overflow`. Длинные названия выталкивают значок совпадения за экран.

### 3. Элементы списка песен в подборке (`pages/collections/[id].vue`)

- `.song-item` без защиты от переполнения. Длинные названия вместе с кнопкой удаления выходят за границы на узких экранах.

### 4. Название подборки в карточке (`components/CollectionCard.vue`)

- `<h3>` без обрезания. Длинные названия сталкиваются с кнопками действий.

---

## План исправлений

### Исправление 1: Навбар (`layouts/default.vue`)

Добавить стили для `#navbar-center`:
```css
#navbar-center {
  flex: 1;
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
  margin: 0 0.5rem;
}
.nav-title {
  font-size: 1.2rem;  /* было 2rem */
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
  display: block;
}
@media (min-width: 640px) {
  .nav-title {
    font-size: 1.5rem;
  }
}
```

Починить конфликт отступов в `.page-content`:
```css
.page-content {
  padding-top: calc(56px + 1rem);
  padding-left: 1rem;
  padding-right: 1rem;
  padding-bottom: 1rem;
  flex: 1;
}
```

### Исправление 2: Поисковая выдача (`pages/index.vue`)

Добавить в `.song-title`:
```css
.song-title {
  flex: 1;
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

Добавить `.score`:
```css
.score {
  flex-shrink: 0;
}
```

### Исправление 3: Список песен подборки (`pages/collections/[id].vue`)

Переделать вёрстку: `.song-item` как flex-строка с обрезанием на ссылке заголовка:
```css
.song-item {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: 0.5rem;
}
.song-item a {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
.remove-btn {
  flex-shrink: 0;
}
```

### Исправление 4: Название подборки в карточке (`components/CollectionCard.vue`)

Добавить:
```css
.collection-info h3 {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
.collection-info {
  flex: 1;
  min-width: 0;
  margin-right: 0.5rem;
}
```
