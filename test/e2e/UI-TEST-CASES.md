# Каталог UI тест-кейсов

Полный перечень пользовательских сценариев для E2E-покрытия.
Реализованные кейсы разнесены по specs/ (по страницам приложения) и
journeys/ (сквозные сценарии). Остальные — спецификация для последующей
реализации.

Условные обозначения:
- ✅ — реализован в Playwright spec
- ⬜ — сформулирован, ждёт реализации

Селекторы централизованы в `lib/selectors.js`, тестовые песни — в
`lib/songs.js` (производны от фикстуры `data/fixtures/songs.fixture.json`).

Соответствие разделов и файлов:

| Раздел | Файл |
|--------|------|
| A. Навигация и каркас | `specs/navbar.spec.js`, `specs/sidebar.spec.js` |
| B. Поиск | `specs/home.spec.js`, `specs/search-layout.spec.js` |
| C. Страница песни | `specs/song.spec.js`, `specs/song-goto.spec.js`, `specs/favorites.spec.js`, `specs/add-to-collection.spec.js` |
| D. Настройки | `specs/settings.spec.js` |
| E. Подборки | `specs/collections.spec.js`, `specs/add-to-collection.spec.js` |
| F. Сайдбар | `specs/sidebar.spec.js` |
| G. Адаптивность | `specs/responsive.spec.js` |
| H. Состояния данных | `specs/home.spec.js` |
| Джорни | `journeys/*.spec.js` |

Селекторы приведены по коду компонентов (`SongSearchInput.vue`,
`pages/song/[number].vue`, `layouts/default.vue` и др.).
